# finalproject

1) Open  PgAdmin
2) Navigate to your localhost and put in the password: root
3) Right-Click on databases and select New Database
4) Enter zoo for the name and postgres for the Owner and hit ok
5) Open Eclipse
6) Select File and chose import
7) Open the maven option
8) Select existing Maven project and hit next
9) Hit Browse and navigate to where you cloned the repository select it and hit finish
10)Open src/main/java, then open com.billyhornfinal.springboot and run Application as a Java application
11)Open an internet browser and navigate to localhost:8080 to see the webApp.
